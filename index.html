Creative Technology

Creative technology (CreaTe) is a bachelor's programme thaught at the University of twente and Vrije Universiteit Amsterdam. The study challenges you to develop innovative applications while using the latest technology. It is a diverse study that teaches students how to think creatively in the process of coming up with onnovative solutions, while also enabling them to execute these solutions into tangible prototypes. Learn more about this bachelor's programme at utwente.nl or vu.nl.

During my short time participating in this programme, I have already completed many projects and worked with a ton of software. all my completed projects can be found on this website.

Fusion 360

Fusion 360 is a computer-aided design (CAD), computer-aided manufacturing (CAM), computer-aided engineering (CAE) and printed circuit board (PCB) design software application. Fusion has built-in capabilities for 3D modeling, simualtion and documentation. Learn more

In the first module of my study programme this software was explained to us during a course in a few tutorials. This are the two assignments from those tutorials.

For the final assignment of that course, we had to create a nutcracker in groups of three. I was assigned to create the body in the shape of a pinguin.

During module 2 of the study programme, I also used the software to create a casing for our project. This model of the casing was then used to lasercut a part of our prototype. Inside this casing, an arduino and breadboard was placed.

Blender

Blender is a open-source 3D computer graphics software tool. It is used for creating animated films, visual effects, art, 3D-printed models, motion graphics, interactive 3D applications and virtual reality. Learn more

During the course of the second module, a course about Blender was taught. So, I have completed three assingments using this software. The first assignment was to import an assignment from Fusion and create an animation with it.

The second assignment was to create a minecraft-like temple on a pedestal. In this process UV-textures and wrapping was used.

The final assignment was to model a japanese house. In this assignment different techniques for modelling and texturing were used.
